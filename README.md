## 📌 Dataset

**Primary dataset:** Telco Customer Churn (BigML)  
- Training set: 2,666 records · 21 features · 14.6% churn rate  
- Test set: 667 records  

**Other datasets:** Boston Housing, AAPL Stock Prices, Iris, Sentiment

---

## 🔑 Key Findings

- International plan holders churn at **4x** the rate of non-holders
- Customers with **4+ service calls** show 48%+ churn probability  
- Total minutes and total charges show **perfect multicollinearity** (r ≈ 1.0)
- Polynomial regression (degree 2) achieved **R² = 0.8056** on house price prediction
- K-Means optimal cluster count confirmed at **K=3** (Silhouette = 0.459)
