## Task 2 — Customer Churn Analysis Dashboard (Power BI)

**Dataset:** Telco Customer Churn (churn-bigml-80.csv)  
**Records:** 2,666 customers · 20 features  
**Tool:** Power BI Desktop

---

### Dashboard Overview

Interactive single-page dashboard with 4 KPI cards, 3 slicers, and 4 visuals
built to communicate churn patterns across customer segments.

---

### KPI Cards

| Metric | Value |
|---|---|
| Total Customers | 2,666 |
| Churn Rate | 14.6% |
| Avg Daily Minutes | 179.5 |
| Avg Service Calls | 1.56 |

---

### Slicers
- Voice Mail Plan
- International Plan
- State

---

### Visuals

| Chart | Type | Key Insight |
|---|---|---|
| Customer Distribution: Churned vs Not Churned | Horizontal Bar | 388 churned vs 2,278 stayed |
| Churn Rate by International Plan | Stacked Bar | Int'l plan holders: 118 churned vs 152 stayed — significantly higher churn ratio |
| Churn by Number of Service Calls | Clustered Bar | Churn spikes sharply after 3+ service calls |
| Day Minutes vs Service Calls | Scatter (bubble) | High daily minutes + high service calls correlates with churn |
